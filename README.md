# NLP Projects

Text data is messy, ambiguous and context-dependent — which makes it the most 
interesting kind of data to work with.

These projects explore how neural networks learn to understand and generate 
human language — from classifying emotions in movie reviews to generating 
new text from scratch.

## Projects

| Project | Architecture | Dataset |
|---------|-------------|---------|
| IMDB Sentiment Analysis | LSTM | IMDB Movie Reviews |
| RNN Text Generation | RNN | Custom text corpus |

## What I learned

Sentiment analysis sounds simple until you realize how much context matters. 
A negative word in a positive sentence completely changes the meaning — and 
teaching a model to understand that is genuinely hard.

Text generation was the fun one. Watching a model slowly learn patterns 
in language and start producing coherent sentences is one of those moments 
that makes you appreciate how language models actually work under the hood.
